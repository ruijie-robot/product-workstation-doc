# 产品工作台 （一期）

  
## 一、产品概述

**产品工作台（一期）** 通过自然语言交互，帮助销售人员快速完成基金筛选与基金绩效分析，生成专业、可视化的产品分析材料用于客户沟通与产品推介。

**需求场景**：

```
销售人员打开 产品工作台

│

├── 场景一：客户想买基金，我需要推荐

│ └── 在 AI 助手中输入：

│ "筛选近1年年化收益8%以上、最大回撤5%以内的广发基金管理的偏股混合基金"

│ → 系统返回筛选结果表格 → 点击某只基金查看深度分析 → 转发报告给客户

│

├── 场景二：客户问"XX基金最近表现怎么样？"

│ └── 在 AI 助手中输入：

│ "帮我分析广发稳健增长A这只基金"

│ → 系统生成深度分析报告（净值曲线、事件区间表现、亮点/不足、同行对比）

│ → 用分析结论回复客户

│

└── 场景三：我想找能涨抗跌的基金推荐给客户

└── 在 AI 助手中输入：

"帮我筛选能涨抗跌的基金"

→ 系统按 upCapture/downCapture 比率筛选 → 返回抗风险能力强的基金列表

```

#### 左侧边栏 （工作区）

**我的笔记**：
这是一个文件夹系统，文件夹系统可以：
1. 创建嵌套文件夹
2. 删除/重命名文件夹
3. 可以拖动文件夹&文件位置
4. 笔记都是.md文件，如果不是.md文件，无法打开
5. 文件都是静态的，多次打开不会刷新变化



**我的看板**
这是一个文件夹系统，里面的文件是[[看板文件]]，文件夹系统可以：
1. 创建嵌套文件夹
2. 删除/重命名文件夹&[[看板文件]]
3. 可以拖动文件夹&[[看板文件]]位置
4. 每次打开[[看板文件]]都会刷新数据


**搜索框**
1. 在“工作区”搜索框内，可以搜索工作区的文件夹/文件/看板
2. 搜索框旁边的+，可以添加基金（eg.添加“ZX策略均衡”文件夹，文件夹里面有默认[[看板文件]]）

### 定义

[[看板文件]]： 
- 看板文件是一个html页面，这个页面只支持三种信息：纯文本，图表（包括折线图，柱状图，饼状图，表格等等）和引用连接（点击引用链接，可以打开引用网页）。
- 看板文件只能存放在“我的看板”文件夹及其嵌套文件夹下
- 看板文件不可以被用户修改
- 看板文件里面的图表，每次打开会刷新到最新数据

[[.md文件]]：
- 从看板文件转为的.md文件，只有三种信息：纯文本，图片和引用链接
- .md文件只能存放在“我的笔记”和“技能”文件夹中，不能放在“我的看板”文件夹里
- 可以从[[标签页]]打开.md文件，以可编辑的preview形式显示.md文件
